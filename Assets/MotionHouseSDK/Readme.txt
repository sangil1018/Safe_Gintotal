SDK 를 설치하신 후 C:\MotionHouse\SDK\lib 폴더의 다음과 같은 파일을 찾아주세요

MotionHouseSDK.ini
DEVICE_SETTING.ini

위의 파일을 Unity 프로젝트 폴더에 복사해주시면 유니티 에디터에서도 실기기 Test 가 가능합니다.
(주의 할점은 Asset 폴더가 아니라 Project 폴더입니다. )


또한 빌드 후에는 위의 파일을 빌드된 프로젝트 exe 실행파일이 존재하는 폴더에 복사해 넣으시면됩니다.

위의 파일 셋팅에 대해서는 

http://www.motionhouse.co.kr/bbs/board.php?bo_table=03_01&wr_id=31 영상의 설명을 참조하시면 되겠습니다.
